Overview

Blip is a simple, interpreted programming language designed for ease of use and flexibility. It supports variables, functions, loops, conditionals, lists, dictionaries, threading, asynchronous calls, and built-in functions for mathematical, list, and HTTP operations. Blip is ideal for quick scripting tasks, API interactions, concurrent programming, and learning programming concepts.

Syntax

Blip uses a straightforward syntax with Python-like semantics. Below are the key elements:

  • Variables: Assign values using =. No type declarations needed.
  • Functions: Defined with func keyword, followed by parameters and a body enclosed in end;.
  • Loops: Supports for and while loops, with break and continue for control flow.
  • Conditionals: Uses if, else, and end; for control flow.
  • Lists: Created with [...], supports indexing (e.g., list[0]) and operations like append and pop.
  • Dictionaries: Created with {key: value}, supports key access via dot notation (e.g., dict.key).
  • Threading: Supports thread creation with thread and asynchronous calls with async, allowing concurrent execution.
  • Comments: Lines starting with # are ignored.
  • Statements: Must end with a semicolon (;).
  • Debug Mode: Enable with {debug} at the start of a script to show execution time and detailed error traces.

Example syntax:


{debug}
x = 10;

func add(a, b)
    return a + b;
end;

for i in range(1, 5)
    print(i);
end;

if x > 5
    print("x is greater than 5");
else
    print("x is 5 or less");
end;

data = {"key": "value"};
print(data.key);

thread add(2, 3);

Built-in Functions

Blip provides a rich set of built-in functions for various tasks:

  • print(arg, ...): Prints arguments to the console, space-separated.
  • input(prompt): Reads user input with an optional prompt.
  • int(x), float(x), str(x): Type conversions.
  • abs(x), sqrt(x), sin(x), cos(x), tan(x), asin(x), acos(x), atan(x), atan2(y, x), sinh(x), cosh(x), tanh(x): Mathematical functions.
  • log(x), log10(x), log2(x), exp(x): Logarithmic and exponential functions.
  • floor(x), ceil(x), round(x): Rounding functions.
  • max(arg, ...), min(arg, ...): Maximum and minimum values.
  • pow(x, y): Power function (x raised to y).
  • range(start, end, step): Generates a list of numbers.
  • append(lst, item), pop(lst, index): List operations to add or remove elements.
  • sort(lst), reverse(lst): Sort or reverse a list.
  • len(x), size(x): Returns length of lists or strings.
  • sum(lst), avg(lst): Sum or average of list elements.
  • random(), randint(a, b): Random number generation.
  • factorial(n), gcd(a, b), lcm(a, b), is_prime(n), fib(n): Number theory functions.
  • mod(x, y), div(x, y): Modulo and integer division.
  • pi(), e(), deg(x), rad(x): Mathematical constants and conversions.
  • get(url, headers): Sends HTTP GET request, returns dictionary with status, content, json, and error keys.
  • post(url, data, headers): Sends HTTP POST request with JSON data, returns dictionary with status, content, json, and error keys.
  • thread_create(fn, arg, ...): Creates a thread to run function fn with arguments, returns thread ID.
  • thread_join(tid): Waits for thread with ID tid to complete, returns result or error.
  • thread_id(): Returns the current thread's ID.
  • thread_lock(), thread_unlock(): Acquires or releases a global thread lock.
  • async_call(fn, arg, ...): Asynchronously calls function fn with arguments, returns thread ID.
  • get_result(tid): Retrieves result of async call with thread ID tid.
  • type(x): Returns the type name of x.

For a full list, refer to the interpreter's b dictionary in the source code.

Examples

Fibonacci Sequence


func fib(n)
    if n <= 0
        return 0;
    end;
    if n == 1
        return 1;
    end;
    a = 0;
    b = 1;
    for i in range(2, n + 1)
        temp = a + b;
        a = b;
        b = temp;
    end;
    return b;
end;

print(fib(10));

Factorial Calculation


func factorial(n)
    if n <= 1
        return 1;
    end;
    return n * factorial(n - 1);
end;

print(factorial(5));

List Operations


numbers = [1, 2, 3];
append(numbers, 4);
print(numbers);
print(numbers[2]);
print(numbers.length);

HTTP Requests and Dictionaries


response = get("https://jsonplaceholder.typicode.com/todos/1");
print(response.status);
print(response.json);
data = {"title": "foo", "body": "bar", "userId": 1};
response = post("https://jsonplaceholder.typicode.com/posts", data);
print(response.status);
print(response.json);

Threading Example


func compute_sum(a, b)
    thread_lock();
    print("Computing sum in thread ", thread_id());
    thread_unlock();
    return a + b;
end;

tid = thread_create(compute_sum, 5, 10);
result = thread_join(tid);
print("Thread result: ", result);

Asynchronous Call Example


func long_task(x)
    thread_lock();
    print("Starting long task in thread ", thread_id());
    thread_unlock();
    return x * x;
end;

tid = async_call(long_task, 4);
print("Async task started with thread ID: ", tid);
result = get_result(tid);
print("Async result: ", result);
